# Shortly Tokens

This is just an example of design tokens.

**JSON** · **SCSS** · **CSS**

Design tokens for Shortly

- colors
- typography
- borders
- z-index

## Setup

Create an `.env` with the following variables:

```
FIGMA_TOKEN='FIGMA TOKEN'
FIGMA_FILE_ID='FIGMA FILE ID'
FIGMA_FILE_NODE_ID='FIGMA FILE NODE ID'
```

## Sync tokens

```
$ npm run tokens:sync
```
